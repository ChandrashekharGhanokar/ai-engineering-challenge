Key Concepts
  • Embeddings map text into a high-dimensional vector space where semantic similarity = geometric proximity.
  • nomic-embed-text produces 768-dimensional float32 vectors — runs fully offline via Ollama.
  • Cosine similarity measures the angle between two vectors: dot(a,b) / (|a|·|b|). Range: −1 to 1.
  • Scores ≥ 0.75 = strong semantic match  |  ≥ 0.50 = related  |  < 0.50 = weak/unrelated.
  • The heatmap visualises the raw activation pattern of the 768-dim vector (first 512 shown).
  • In production, vectors are pre-computed and stored in a vector DB (pgvector, Qdrant, Weaviate).
